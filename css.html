O que é CSS?

O CSS (Folha de Estilo em Cascata) é o código que você usa para dar estilo à sua página Web.

Explicação e demonstração de seletores básicos, propriedades, valores, classes, IDs, cascata e especificidade.

Seletores de Tag: Aplicam estilos a todos os elementos de um tipo específico.

Seletores de Classe: Aplicam estilos a todos os elementos com uma classe específica.

Seletores de ID: Aplicam estilos a um único elemento com um ID específico.

Propriedades e Valores: Definem o estilo e a aparência dos elementos HTML. Exemplos comuns incluem color, font-size, margin, padding, e background-color.

Cascata: Ordem de aplicação das regras CSS, com regras mais específicas e recentes tendo prioridade.

Especificidade: Calcula qual regra aplicar quando várias regras são aplicáveis, baseando-se na combinação de tipos de seletores.
