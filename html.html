O que é HTML?

HTML (Linguagem de Marcação de Hipertexto) é o código que você usa para estruturar uma página web e seu conteúdo. Por exemplo, o conteúdo pode ser estruturado em parágrafos, em uma lista com marcadores ou usando imagens e tabelas.

O que são as tags 'p', 'ul', 'ol', 'li', 'table', 'img' e 'a'?

A tag 'p' no HTML é usada para definir um parágrafo de texto. É uma das tags mais básicas e essenciais para a estruturação de conteúdo textual em uma página da web.

ul (Unordered List) Utilizada para criar uma lista não ordenada, onde os itens são geralmente marcados com pontos.

ol (Ordered List) Utilizada para criar uma lista ordenada, onde os itens são numerados.

li (List Item) Utilizada para definir um item dentro de uma lista ul ou ol.

A tag 'table' é usada para criar tabelas na página da web. Uma tabela é uma estrutura que organiza dados em linhas e colunas.

A tag 'img' é usada para inserir imagens em uma página da web. É um elemento auto-fechado, o que significa que não precisa de uma tag de fechamento.

A tag 'a' é usada em combinação com o atributo href, que especifica o destino do link. É fundamental para a navegação e interatividade na web, e seu uso adequado é essencial para criar uma experiência de usuário eficiente e amigável.
